# Tech Week — UniCesumar Londrina

## About
Full-stack application for UniCesumar Londrina's Tech Week event.
Frontend built with Next.js 16 App Router, TypeScript, Tailwind CSS and shadcn/ui.
Backend is a separate repository (tech-week-api) built in Rust with Axum and SQLite.

## Running locally
npm run dev

## Conventions
- Commits in English following conventional commits (feat:, fix:, chore:, style:, etc)
- Components in src/components
- Pages in src/app following App Router structure
- Brand colors via --brand CSS variable defined in globals.css
- Mock data lives in src/lib/api.ts — set USE_MOCK to false when backend is ready

## Do not touch
- src/components/ui/ — generated by shadcn/ui, do not edit manually
- src/lib/utils.ts — used internally by shadcn/ui
